赛程设置
  初赛：大赛主办方邀请评审专家共同组成评审小组，对照推荐条件对候选项目及参赛团队进行资格审查，审核通过，且进入小组前50名的项目，进入复赛。
  复赛：（1）复赛通过官网展示、网络投票/点赞指数征集、专家评审等环节进行；（2）线上投票/点赞指数占总成绩10%，专家评审占90%；（3）线上线下综合评分前15名（创业类、创意类分别）入围决赛。
  决赛：采取现场评比打分的方式进行。现场评比分为项目路演和专家提问两个环节，其中项目路演PPT/VCR展示8分钟，评委提问5分钟。
